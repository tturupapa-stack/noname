# Supervisor Report 스킬

## 개요

멀티에이전트 워크플로우 완료 후 최종 리포트를 생성합니다. 사용자에게 작업 결과를 명확하게 전달하기 위한 표준화된 리포트 형식을 제공합니다.

## 사용 시점

1. Supervisor가 모든 에이전트 작업 완료를 확인한 후
2. code-reviewer의 품질 리뷰가 완료된 후
3. 사용자에게 최종 결과를 보고할 때

## 리포트 저장 위치

```
output/reports/{YYYY-MM-DD-task-name}/final-report.md
```

## 리포트 구성

### 필수 섹션

1. **요청 사항**: 사용자의 원본 요청
2. **수행된 작업**: 에이전트별 작업 내용
3. **테스트 결과**: test-runner 결과
4. **코드 리뷰 결과**: code-reviewer 판정
5. **결과물 위치**: 생성된 파일 경로

### 선택 섹션

- **다음 단계**: 향후 작업 권고 (필요시)
- **이슈 및 제한사항**: 알려진 문제점
