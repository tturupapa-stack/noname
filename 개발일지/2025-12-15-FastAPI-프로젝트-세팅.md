# 개발일지 - FastAPI 프로젝트 세팅

**작성 시각**: 2025-12-15 (초기 세팅)

## 해결하고자 한 문제

백엔드 API 서버 구축을 위한 FastAPI 프로젝트 초기 세팅
- 프론트엔드(Next.js)와 연동 가능한 백엔드 서버 필요
- Yahoo Finance 데이터 조회를 위한 yahooquery 라이브러리 통합

## 해결된 것

✅ 프로젝트 디렉토리 구조 생성
```
/backend
  /api          - 라우터 (API 엔드포인트)
  /services     - 비즈니스 로직
  /models       - Pydantic 모델
  main.py       - FastAPI 앱 엔트리포인트
  requirements.txt
```

✅ FastAPI 앱 설정 (main.py)
- 기본 앱 인스턴스 생성
- CORS 미들웨어 설정 (localhost:3000 허용)
- 헬스체크 엔드포인트 (`/`, `/health`)

✅ 의존성 패키지 정의 (requirements.txt)
- fastapi>=0.115.0
- uvicorn[standard]>=0.32.0
- yahooquery>=2.3.7
- pydantic>=2.0.0

✅ 서버 실행 테스트 완료
- `GET /` → `{"message":"당신이 잠든 사이 API","status":"running"}`
- `GET /health` → `{"status":"healthy"}`

## 해결되지 않은 것

⚠️ 실제 API 라우터 구현 필요 (주식 데이터 조회 등)
⚠️ Pydantic 모델 정의 필요
⚠️ 비즈니스 로직 서비스 구현 필요

## 향후 개발을 위한 컨텍스트 정리

### 서버 실행 방법
```bash
cd backend
uvicorn main:app --reload --port 8000
```

### API 문서
서버 실행 후:
- Swagger UI: http://localhost:8000/docs
- ReDoc: http://localhost:8000/redoc
